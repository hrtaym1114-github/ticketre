# 🎫 チケトレ (TickeTre)

チケットぴあでのチケット購入をアシストするChrome拡張機能です。

## 主な機能

- **sorryページ自動リトライ** — アクセス集中ページを検知して自動で再アクセス（ランダム間隔）
- **LP同意画面の自動通過** — 同意チェックボックス＆ボタンを自動クリック
- **座席ハイライト** — 空席(○/◎)を緑、残りわずか(△)をオレンジで強調表示
- **ログイン自動入力** — 保存した認証情報を自動入力
- **音声アラート** — 購入画面到達時にビープ音で通知
- **戦況モニター** — サイドパネルでリアルタイム状況表示

## インストール方法

1. このリポジトリをクローンまたはダウンロード
2. Chromeで `chrome://extensions` を開く
3. 右上の「デベロッパーモード」をON
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. `ticketre` フォルダを選択

> Brave, Edge, Vivaldi など Chromium系ブラウザでも同じ手順で使えます。

## 使い方

1. 拡張機能アイコンをクリックしてポップアップを開く
2. ぴあ会員ID・パスワードを入力（ローカル保存のみ、外部送信なし）
3. ターゲット公演のURL・優先度を設定
4. 「💾 保存」→「🚀 発射」

### ページ状態と動作

| 状態 | アイコン | 動作 |
|------|---------|------|
| LP同意画面 | 📋 | 自動でチェック＆通過 |
| sorryページ | ❌ | 設定間隔で自動リトライ |
| 待機画面 | ⏳ | リロードせず自動遷移を監視 |
| 発売前 | ⏰ | 30秒間隔でリトライ |
| 購入画面 | 🎉 | 音声アラート＋座席ハイライト |
| 完売 | 🔴 | 次の公演に自動切替 |

## 注意事項

- ぴあは1アカウント1セッション制限があるため、**1ブラウザ1公演**で使用してください
- 複数公演を同時に攻めたい場合は、別ブラウザ（Chrome + Brave等）で別アカウントを使用
- 認証情報はブラウザのローカルストレージにのみ保存されます

## 技術仕様

- Chrome Extension Manifest V3
- Service Worker によるタブ状態管理
- MutationObserver による動的DOM変更監視
- Web Audio API による音声アラート
